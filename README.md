# Bitcoin Market Sentiment vs Trader Performance Analysis

## Objective

Analyze the relationship between Bitcoin Fear & Greed Index and Hyperliquid trader performance.

## Dataset

- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trading Data

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Workflow

1. Data Cleaning
2. Data Merging
3. Feature Engineering
4. Exploratory Data Analysis
5. Business Insights
6. Recommendations

## Results

- Fear had the highest trading activity.
- Extreme Greed showed the highest average profit and win rate.
- @107 generated the highest cumulative profit.
- Trading activity peaked around 7–8 PM IST.
